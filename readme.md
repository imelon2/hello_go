## Hello Go

#### `./bin` : install | build된 실행 가능한 파일 폴더
#### `./pgk` : package 파일 폴더
#### `./src` : source 저장 폴더
> default path : ~/go/pkg


## go 환경설정(env) 조회 및 설정
1. `go env` 입력
    ```
    GOBIN=''
    GOPATH='/Users/choi/go'
    ```

## go 명령어
1. compile 명령어
run : code 즉시 실행
build : 실행 가능한 파일로 빌드
install : 디렉토리 내의 코드를 최종적인 실행 가능한 파일로 빌드
